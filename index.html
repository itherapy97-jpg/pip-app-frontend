Peace-In Practice™

Intuit Best Friend

Intuit Best Friend Foundation (Non-Profit)

"내면의 평화는 고치는 것이 아니라,
이미 있는 것을 발견하는 것입니다."

Peace-In Practice™는 기존 CBT(인지행동치료)와 정반대 방향에서 접근하는 인류 최초의 독자적 내면 교육 방법론입니다.

전문 상담가가 창안한 이 프로그램은 교육 과정 안에 1:1 실제 상담이 정기적으로 포함되어 있습니다.

⚠ 중요 안내

본 프로그램은 전문 상담가의 지도 하에만 효과적으로 작동합니다. 허가된 참여자만 접근 가능합니다.

사용설명서

Peace-In Practice™ 이용 안내

🌿 프로그램이란?

Peace-In Practice™는 전문 상담가가 개발한 내면 교육 프로그램입니다. 앱을 통해 교육 모듈을 진행하고, 상담가와 1:1 상담을 주고받습니다.

본 프로그램은 반드시 상담가의 초대코드를 통해서만 참여할 수 있습니다. 일반 공개 가입은 불가합니다.

🚀 시작하는 방법
1
초대코드 받기
담당 상담가로부터 6자리 초대코드를 전달받으세요. 초대코드가 없으면 참여할 수 없습니다.
2
계정 만들기
'시작하기' 버튼 → '초대코드 (첫 방문)' 탭 선택 → 초대코드 입력 → 이메일과 비밀번호 설정
3
다음 방문부터는 이메일 로그인
'이메일 로그인' 탭에서 등록한 이메일과 비밀번호로 로그인합니다.
📱 주요 기능 안내
⊙ 홈 화면
• 이번 주 진행률 확인
• 배정된 교육 모듈 목록
• 세션 일정 확인 및 요청
• 상담가 답변 알림
◈ 교육 세션
• 상담가가 준비한 교육 모듈 진행
• 객관식·주관식 질문에 답변
• 완료한 모듈은 '완료' 표시
✉ 개인 상담
• 마음속에 있는 내용을 자유롭게 작성
• 작성 내용은 암호화되어 전송됩니다
• 상담가 답변이 오면 홈 화면에 알림 표시
• 답변 확인 후 내용은 화면에서 사라집니다
📅 세션 시간 약속
• 원하는 세션 시간을 선택해 요청
• 상담가 확인 후 최종 확정됩니다
🔒 개인정보 및 보안
✓ 상담 내용은 암호화되어 서버에만 보관됩니다
✓ 작성 내용은 이 기기에 저장되지 않습니다
✓ 상담가 이외의 제3자는 내용을 볼 수 없습니다
✓ AI 번역기 등 외부 서비스에 내용이 전송되지 않습니다
❓ 자주 묻는 질문
초대코드를 잃어버렸어요.
담당 상담가에게 연락하여 코드를 다시 받으세요. 한 번 사용된 코드는 재사용할 수 없습니다.
비밀번호를 잊어버렸어요.
현재 비밀번호 재설정 기능은 담당 상담가를 통해 처리됩니다. 상담가에게 문의해 주세요.
교육 모듈이 보이지 않아요.
모듈은 상담가가 직접 배정합니다. 아직 배정받지 못한 경우 상담가에게 문의해 주세요.
상담 답변은 얼마나 걸리나요?
담당 상담가의 일정에 따라 다르며, 일반적으로 24~48시간 이내에 답변이 전송됩니다.
이 앱은 어떤 언어로 사용할 수 있나요?
이 안내 페이지는 상단의 🌐 언어 버튼으로 원하는 언어로 번역할 수 있습니다. 앱 내 교육·상담은 담당 상담가가 설정한 언어로 진행됩니다.
